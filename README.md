# 🚀 Streamlit Basics

A beginner-friendly repository that demonstrates the fundamentals of **Streamlit**, a powerful Python framework for building interactive web applications and dashboards with minimal code.

This project is designed for students, developers, data analysts, and AI enthusiasts who want to learn how to create beautiful web apps using Python without requiring frontend development skills.

---

## 📌 What is Streamlit?

Streamlit is an open-source Python framework that allows developers to build and deploy interactive web applications for data science, machine learning, AI, and analytics projects quickly and efficiently.

With Streamlit, you can:

* Build web applications using only Python
* Create interactive dashboards
* Visualize data in real-time
* Deploy machine learning models
* Develop AI-powered applications
* Create forms, charts, and widgets with ease

## 🛠️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/streamlit-basics.git
cd streamlit-basics
```

### 2. Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Run the Streamlit application using:

```bash
streamlit run app.py
```

After execution, Streamlit will automatically open the application in your browser.

If it doesn't open automatically, visit:

```text
http://localhost:8501
```
